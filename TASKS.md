# Legal RAG Agent 求职导向任务清单

- 大模型应用开发

暂不以算法研究、模型训练、AI Infra 为主要目标。

建议执行顺序：

```text

→ 数据与引用
→ 评测
→ 工具层
→ Agent Harness
→ Trace
→ Docker
→ 简历展示
```
